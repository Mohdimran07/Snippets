# Snippets Application

This is a **Snippets Application** built using [Next.js 15](https://nextjs.org/), designed for performing **CRUD (Create, Read, Update, Delete)** operations. The application serves as a foundation for managing data with an intuitive user interface and efficient backend logic.

---

## Features

- **Create**: Add new snippets with relevant details.
- **Read**: View all stored snippets in a user-friendly interface.
- **Update**: Edit existing snippets to keep them updated.
- **Delete**: Remove snippets no longer needed.
- Built using the latest features of **Next.js 15**.
- Fully responsive design for mobile, tablet, and desktop.

---

## Technologies Used

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Styling**: Tailwind CSS
- **Database**: Prisma with sqlite
- **API**: Next.js API Routes for seamless server-side logic.
